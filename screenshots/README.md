# Screenshots

This directory contains screenshots of Void OS running in QEMU.

## Screenshot Files

- **boot-screen.png** - Initial boot screen showing memory allocator tests
- **shell-help.png** - Interactive shell with help command output
- **meminfo.png** - Memory information display using `meminfo` command

## How to Add Screenshots

1. Take a screenshot of Void OS running in QEMU
2. Save it as PNG format in this directory
3. Update `README.md` to reference the new screenshot
4. Use descriptive filenames (e.g., `graphics-demo.png`, `cpuinfo-output.png`)

## Recommended Screenshots

- Boot screen
- Shell with help command
- Memory information (`meminfo`)
- CPU information (`cpuinfo`)
- Graphics mode demo
- Test suite output
